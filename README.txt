Steps to Set Up Application:

1. Open pgAdmin4 and create a Database called A3Students
2. Open the scripts folder and using the Initialization SQL create the table in your database
3. Open the scripts folder and insert the data using Data SQL into your database 
4. Open source_code and replace username and password with your pgAdmin username and password
5. Run the application, all the options are predetermined and will update both in pgAdmin and will be displayed in the console
